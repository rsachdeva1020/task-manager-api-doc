Task Manager API

Path : https://rohan-task-managerapp.herokuapp.com

Create User(SignUp)

Request Type : POST

/users 
      body : {  
          "name" : "Rohan",
          "email" : "rohan@gmail.com", 
          "password" : "password@123" 
      }

Will receive Authorization Token (to be used in future) and details of the user Created.

Login User

Request Type : POST

/users/login 
      body : { 
          "email" : "rohan@gmail.com",
          "password" : "password@123" 
      }

Will receive Authorization Token (to be used in future) and details of the user LoggedIn.

Create Tasks

Request Type : POST

/tasks Authorization : Bearer {token} 
      body : { 
          "description" : "Data Structures",
          "completed" : false 
      }

Reading all tasks

Request Type : GET

/tasks

Options / Params

  • sortBy=createdAt:{desc or asc}
  • completed={false or true}
  • limit={number of results}
  • skip={number of tasks}

In Header : Authorization : Bearer {token}

Read own profile

Request Type : GET

/users/me

In Header : Authorization : Bearer {token}

Reading specific task

Request Type : GET

/tasks:{id}

In Header : Authorization : Bearer {token}
